Guia completo para uso correto da marca

Versão 1.0 \| 2025

**Sumário**

**1. Sobre a Marca**

A Emplo.ia é uma plataforma de tecnologia orientada a inteligência
artificial. Sua identidade visual foi projetada para transmitir
modernidade, confiança e precisão --- qualidades essenciais para uma
marca que atua na interseção entre tecnologia e relações humanas no
trabalho.

Este manual reúne todas as diretrizes necessárias para a correta
aplicação da identidade visual da Emplo.ia em materiais institucionais,
digitais e promocionais. Seu objetivo é garantir consistência e
coerência em qualquer contexto de uso.

**2. Logotipo**

**2.1 Versões do Logotipo**

A marca Emplo.ia deve ser utilizada somente nas versões oficiais
apresentadas neste manual. Para garantir consistência visual, não altere
as formas, cores ou proporções do logotipo.

**2.3 Área de Segurança**

A área de proteção existe para assegurar a clareza visual e a correta
leitura da marca. Ela impede que outros elementos visuais ou textos se
aproximem excessivamente do logotipo, comprometendo sua identificação.

Regra: nenhum elemento gráfico ou tipográfico deve invadir o espaço de
proteção ao redor da marca. A área mínima de respiro corresponde à
altura da letra \"E\" do símbolo em todos os lados.

**2.4 Tamanho Mínimo**

  ------------------ ----------------------------------------------------
  **Digital**        Largura mínima de 120 px para o logotipo completo;
                     32 px para o símbolo isolado.

  ------------------ ----------------------------------------------------

  ------------------ ----------------------------------------------------
  **Impresso**       Largura mínima de 30 mm para o logotipo completo; 8
                     mm para o símbolo isolado.

  ------------------ ----------------------------------------------------

**3. Aplicação em Fundos**

**3.1 Fundos Claros**

O logotipo pode ser aplicado em suas cores institucionais sobre fundos
claros e uniformes, desde que esses fundos não ultrapassem 20% da
intensidade total de cor. Sempre que houver condições adequadas, a
versão colorida deve ser a escolha principal.

+-----------------------------------------------------------------------+
| **Fundos compatíveis com a versão colorida:**                         |
|                                                                       |
| -   Branco (#FFFFFF)                                                  |
|                                                                       |
| -   Tons de cinza muito claro (até 20% de saturação)                  |
|                                                                       |
| -   Beige suave --- #E8DAB2                                           |
|                                                                       |
| -   Lilás pastel --- #CBC5EA                                          |
|                                                                       |
| -   Rosa muito claro --- #FAD5D1                                      |
|                                                                       |
| -   Turquesa pastel --- #97EAD2                                       |
+-----------------------------------------------------------------------+

**3.2 Fundos Escuros e Saturados**

Em superfícies escuras ou com alto nível de saturação, utilize as
versões em preto e branco do logotipo, dando preferência à versão
negativa (branca) para garantir contraste suficiente e boa leitura.

+-----------------------------------------------------------------------+
| **Nesses contextos, use logotipo BRANCO (negativo):**                 |
|                                                                       |
| -   Roxo principal --- #3F1757                                        |
|                                                                       |
| -   Fundo preto ou carvão --- #1E1E1E                                 |
|                                                                       |
| -   Qualquer cor com mais de 50% de saturação                         |
|                                                                       |
| -   Fotografias ou imagens de fundo                                   |
|                                                                       |
| -   Gradientes escuros                                                |
+-----------------------------------------------------------------------+

**6. Uso Correto e Uso Indevido**

Não são permitidas aplicações da marca Emplo.ia que desconsiderem as
diretrizes definidas neste manual. A seguir, estão listadas as práticas
corretas e os erros mais comuns a evitar.

**6.1 O que FAZER**

  ----------- -----------------------------------------------------------
  **✔ FAZER** Usar o logotipo apenas nas versões e variações de cor
              aprovadas neste manual.

  **✔ FAZER** Manter as proporções originais do logotipo ao
              redimensioná-lo.

  **✔ FAZER** Respeitar a área de segurança ao redor da marca em todas as
              aplicações.

  **✔ FAZER** Usar a versão branca (negativa) em fundos escuros ou
              saturados.

  **✔ FAZER** Usar a versão colorida como escolha principal sempre que o
              fundo permitir.

  **✔ FAZER** Utilizar o logotipo reduzido (símbolo) apenas em ícones,
              avatares e favicons.

  **✔ FAZER** Consultar a equipe de Marketing antes de qualquer aplicação
              não prevista neste manual.
  ----------- -----------------------------------------------------------

**6.2 O que NÃO FAZER**

  ----------- -----------------------------------------------------------
  **✘ NÃO     Alterar as cores do logotipo para qualquer cor não aprovada
  FAZER**     neste manual.

  **✘ NÃO     Distorcer, esticar ou comprimir o logotipo horizontal ou
  FAZER**     verticalmente.

  **✘ NÃO     Rotacionar ou inclinar o logotipo.
  FAZER**     

  **✘ NÃO     Aplicar sombras, contornos, efeitos de gradiente ou
  FAZER**     texturas sobre o logotipo.

  **✘ NÃO     Separar o símbolo do wordmark em contextos que exigem o
  FAZER**     logotipo completo.

  **✘ NÃO     Inverter a ordem dos elementos (wordmark à esquerda do
  FAZER**     símbolo).

  **✘ NÃO     Usar o logotipo abaixo do tamanho mínimo recomendado.
  FAZER**     

  **✘ NÃO     Colocar o logotipo sobre fundos complexos, fotografias ou
  FAZER**     padrões sem garantir contraste adequado.

  **✘ NÃO     Recriar ou redesenhar o logotipo a partir de elementos
  FAZER**     genéricos.

  **✘ NÃO     Usar a marca fora de sua área de proteção, com outros
  FAZER**     elementos invadindo o espaço de segurança.
  ----------- -----------------------------------------------------------
